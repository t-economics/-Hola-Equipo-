## 🚦 Flujo de Trabajo

```bash
1. Crear rama desde develop o main
2. Nombrar la rama correctamente:
   feature/nombre-funcionalidad
   fix/descripcion-error
3. Hacer commits claros
4. Crear Pull Request
5. Code Review obligatorio
6. Merge tras aprobación ✅
